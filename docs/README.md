# ShadowClaude Documentation

Welcome to the ShadowClaude documentation!

## Overview

ShadowClaude is a powerful AI programming assistant framework built with Rust and Python, providing a development experience comparable to Claude Code.

## Getting Started

```bash
# Install
pip install shadowclaude

# Configure
export ANTHROPIC_API_KEY=your_key

# Run
shadowclaude
```
