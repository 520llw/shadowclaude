# ShadowClaude Documentation (한국어)

ShadowClaude에 오신 것을 환영합니다 - 강력한 AI 프로그래밍 어시스턴트!

## 소개

ShadowClaude는 Rust와 Python으로 구축된 AI 프로그래밍 어시스턴트 프레임워크로, Claude Code와 맞먹는 개발 경험을 제공합니다.

### 주요 기능

- 🚀 **고성능**: Rust 코어, 번개 같은 실행 속도
- 🧠 **지능형 메모리**: 3계층 메모리 시스템
- 🤖 **멀티 에이전트**: Coordinator + Swarm 협력 모드
- 🐱 **BUDDY**: 사이버 펫 시스템
- 👻 **Undercover**: 은밀한 보조 모드
- 🔧 **40+ 도구**: 완전한 개발 툴체인
- 🔒 **보안**: 6계층 권한 방어

## 빠른 시작

```bash
# 설치
pip install shadowclaude

# 설정
export ANTHROPIC_API_KEY=your_key

# 실행
shadowclaude
```
