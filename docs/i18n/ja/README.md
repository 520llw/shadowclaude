# ShadowClaude Documentation (日本語)

ShadowClaude へようこそ - 強力な AI プログラミングアシスタント！

## 概要

ShadowClaude は Rust と Python で構築された AI プログラミングアシスタントフレームワークで、Claude Code に匹敵する開発体験を提供します。

### 主な機能

- 🚀 **高性能**: Rust コア、稲妻のような実行速度
- 🧠 **インテリジェントメモリ**: 3層メモリシステム
- 🤖 **マルチエージェント**: Coordinator + Swarm 協調モード
- 🐱 **BUDDY**: サイバーペットシステム
- 👻 **Undercover**: ステルスアシストモード
- 🔧 **40+ ツール**: 完全な開発ツールチェーン
- 🔒 **セキュリティ**: 6層の権限防御

## クイックスタート

```bash
# インストール
pip install shadowclaude

# 設定
export ANTHROPIC_API_KEY=your_key

# 実行
shadowclaude
```
