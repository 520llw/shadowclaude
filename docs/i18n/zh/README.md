# ShadowClaude 文档 (中文)

欢迎使用 ShadowClaude - 强大的 AI 编程助手！

## 简介

ShadowClaude 是一个基于 Rust 和 Python 构建的 AI 编程助手框架，提供与 Claude Code 相媲美的开发体验。

### 主要特性

- 🚀 **高性能**: Rust 核心，闪电般的执行速度
- 🧠 **智能记忆**: 三层记忆系统，理解你的项目
- 🤖 **多 Agent**: Coordinator + Swarm 协作模式
- 🐱 **BUDDY**: 赛博宠物陪伴系统
- 👻 **Undercover**: 卧底模式，静默协助
- 🔧 **40+ 工具**: 完整的开发工具链
- 🔒 **安全**: 六层权限纵深防御

## 快速开始

```bash
# 安装
pip install shadowclaude

# 配置
export ANTHROPIC_API_KEY=your_key

# 运行
shadowclaude
```
